Learn to Code!

Below find links to sites that provide instruction and tutorial on a wide range of coding languages for all levels of learners.

Swift Playgrounds

An app available for mobile apple products. It's a gamified tutorial that teaches students to write the swift programming language for developing apps and games.

Codeschool

Great tutorials with fun videos that guide students through writing code while teaching syntax for many different languages.

Codecademy

A completely free and also has tutorials in several languages and their syntax, as well as skills and programs that can help students with skills. Codecademy now offers a teacher dashboard.

Dash

By General Assembly. Offers great, completely free simple courses on HTML5, CSS, and Javascript. These languages are free, and offer guidance on correct syntax and coding.

CodeHS

offers free introductory courses in several languages, and has courses aligned to the CSTA standards for middle and high school, along with courses that meet the requirements for AP CS Principles and AP CS.

Khan Academy

Has kid-friendly projects to help students learn the basics of HTML, CSS, SQL, and more advanced Javascript. The students are given documentation and examples, but this has less guidance with syntactical errors.

Beyond Drag & Drop

Teaching studens to write code on their mobile device.

Access the Sway presentation here

Pencil Code

     

Blend Drag & Drop with writing code. Learn HTML, CSS, Javascript, and coffeescript to create professional music, games, stories, or the next viral app!

Scratch

     

Open-Source drag & drop coding. Leatn the structure and logic of a programming language. Share and learn from other students by exploring and remixing projects.

Collaboration Resources

  • Github - All open source, a repository that allows people to copy, share, compare, and merge projects with collaborators.
  • World Wide Web Consortium (W3C)- The independent organizations that helps develop the standards for the world wide web.
  • Mozilla Developer Network- Where experts go to learn and communicate.
  • Stack Overflow- If someone has tried it, or has asked about it, it's likely posted here.
  • W3Schools- Has some good tutorials and examples.

Debugging Resources

    There are a number of software programs students can use to write code it called Integrated Development Environments (IDEs). They provide tools to help locate errors in many coding languages. Some examples are;
  •    Eclipse- Used for a vast number of languages.
  •    Komodo- Good for beginners.
  •    Atom- Easy to use with Github.
  •    And so many more..

    • You can also use an online IDEs that you don't have to download a program for, though they often have less debugging capabilities.

  •    Codepen- for front end (visual and website) languages.
  •    Eclipse che- An online version of eclipse.
  •    Cloud 9
  •    codeanywhere

    • W3C also offers debugging tools that allow you to upload or cut and paste your code for help with debugging

  •    https://www.w3.org/developers/tools/
